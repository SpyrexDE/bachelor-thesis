# Task Granularity in Multi-Agent Systems for Multimodal Artifact Sets

Bachelor thesis repository: **Fabian Keßler**, JMU Würzburg (Games Engineering,
Prof. Dr. Sebastian von Mammen) in cooperation with **Henkel dx** (Georg Boegerl).
*19 June – 10 Sep 2026.*

> **Research question:** When an LLM multi-agent system turns one marketing
> brief into a multimodal artifact set (text + image): does splitting the work
> across agents lose set coherence, do explicit coordination mechanisms restore
> it, at what cost (communication tax, latency), and which topology
> (Monolithic, Coarse, Fine) fits which use case?
